📋 Catalogue Editor

GonnaOrder Scripts

Load a clone store's catalogue, edit price / discount / visibility per item, then apply individually or in bulk.

🔒 Credentials are sent over HTTPS to the serverless function, used once per Load to grab a session token, then discarded. Your email is remembered in this browser for next time; your password is not. If GonnaOrder starts rejecting calls mid-session, hit Load again for a fresh token.
⚠ This looks like a parent catalogue, not a clone. The Apply / Show / Hide buttons use GonnaOrder's /offer/override endpoint, which only works on inherited (clone) catalogues. Editing the parent directly needs a different endpoint that this tool doesn't call yet — applying changes from here will fail. You can still browse the catalogue read-only; to actually push changes, either edit the parent in the GonnaOrder admin directly, or load a clone store's ID instead.
