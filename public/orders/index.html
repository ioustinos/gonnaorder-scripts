📦 Orders Export

GonnaOrder Scripts

Pull all orders in a date window from a GonnaOrder store and export them to CSV.

Defaults to https://admin.gonnaorder.com. Override only for a non-prod instance.
🔒 Credentials are sent over HTTPS to our serverless function, which uses them once to get a session token from GonnaOrder. They are not stored on the server. Your email is remembered in this browser; your password is not.
The window filters on each order's wish time (the requested delivery / pickup time), not its creation time — that's the only date filter GonnaOrder's order-search API exposes. The window is inclusive and interpreted in UTC (from 00:00:00 → to 23:59:59).
Tick the statuses to include. All ticked (or none) = all statuses. A subset is fetched one status at a time and merged. isReady isn't filtered, so both ready and not-ready orders are included.
