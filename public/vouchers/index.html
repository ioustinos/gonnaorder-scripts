🎫 Voucher Importer

GonnaOrder Scripts

Bulk-create vouchers in a GonnaOrder store from a CSV or Excel file.

The numeric ID from admin.gonnaorder.com.
Defaults to https://admin.gonnaorder.com. Override only if you're pointing at a non-prod instance.
🔒 Credentials are sent over HTTPS to our serverless function, which uses them once to get a session token from GonnaOrder. They are not stored on the server. Your email is remembered in this browser for next time; your password is not. 

    


    

      
      
      
Columns: code, discount, discountType (PERCENTILE or MONETARY), type (MULTI_USE or ONE_TIME_USE), orderMinAmount, startDate, endDate, isActive, externalId. Only code and discount are required.
